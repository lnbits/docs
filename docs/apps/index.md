# Companion Apps

> Standalone applications that work alongside your LNbits instance to extend its reach.

Companion apps are separate projects that connect to LNbits via its API. They are not extensions - they run as their own applications and talk to your LNbits server.

## Available Apps

| App | Description | Platform |
|---|---|---|
| [TPoS Wrapper](/apps/tpos-wrapper) | Accept credit cards, debit cards, Google Pay, and Apple Pay via Stripe alongside Lightning | Android |

::: tip Building a companion app?
If you have built an app that integrates with LNbits, reach out on [Telegram](https://t.me/lnbits) to get it listed here.
:::
