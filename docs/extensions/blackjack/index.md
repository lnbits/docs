## Overview

BlackJack brings the classic card game to LNbits. Players wager sats on hands of blackjack against an automated dealer. All bets and payouts are handled via Lightning payments.

## How It Works

1. Create a blackjack table with a configured wallet
2. Share the public game link with players
3. Players place bets in sats, play hands, and winnings are paid out automatically

## Features

- Classic blackjack rules
- Lightning-based wagering
- Shareable game links

::: warning
Gambling is dangerous. Gamble responsibly. [gambleaware.org](https://www.gambleaware.org)
:::

## Setup

1. Enable the extension from the LNbits **Extensions** page
2. Create a new blackjack table
3. Share the game URL with players
