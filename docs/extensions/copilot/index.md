## Overview

Streamer Copilot creates a browser-based overlay for OBS (Open Broadcaster Software) that displays Lightning tips on screen during livestreams. Supports LNURLp comments for paid on-screen messages, an optional troll box, a BTC price ticker, and rearrangeable screen elements.

## Features

- **OBS integration** — connect via browser source in OBS
- **Paid comments on screen** — LNURLp comments displayed as tips
- **Troll box** — optional live chat overlay
- **BTC price ticker** — live Bitcoin price on screen
- **Rearrangeable elements** — drag and drop screen components
- **Webhook triggers** — trigger external events on tips

## Setup

1. Enable the extension from the LNbits **Extensions** page
2. Create a new Copilot and configure your settings
3. Add the browser source URL to OBS
4. Arrange the overlay elements on screen

## Use Cases

- **Livestreaming** — show Lightning tips on screen during streams
- **Podcasting** — display live donations during recordings
- **Events** — live tip feed at conferences or meetups
