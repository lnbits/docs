## Overview

Scrum provides a basic project management board where tasks can have Lightning payments attached. Organize work with a scrum-style workflow and optionally pay for task completion via Lightning.

## Features

- **Task board** — organize tasks in a scrum-style layout
- **Paid tasks** — attach Lightning payments to task completion
- **Simple interface** — lightweight project management

## Setup

1. Enable the extension from the LNbits **Extensions** page
2. Create a new board and add tasks
3. Optionally attach Lightning payments to tasks
