## Overview

RaiseNow creates fundraising campaigns where donations are collected for multiple participants with a live public leaderboard. When the timer ends, funds are distributed among participants. Perfect for competitions, charity events, and community fundraisers.

## How It Works

1. Create a fundraising event with a deadline
2. Add participants and configure the split
3. Share the public event page
4. Donors send sats — the leaderboard updates in real-time
5. When time expires, funds are distributed automatically

## Features

- **Timed campaigns** — set a deadline for the fundraiser
- **Multiple participants** — add team members or causes
- **Live leaderboard** — public rankings update as donations arrive
- **Automatic distribution** — funds split when the timer ends
- **Public donation page** — shareable link for donors

## Use Cases

- **Sporting events** — fans donate to support their favorite team/player
- **Charity fundraisers** — competitive fundraising between causes
- **Community projects** — crowdfund specific project goals
- **Hackathon prizes** — live donation tracking during events

## Setup

1. Enable the extension from the LNbits **Extensions** page
2. Create a new fundraising event with a deadline
3. Add participants and configure how funds are split
4. Share the event page and watch the leaderboard
