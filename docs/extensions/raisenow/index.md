## How it works

Create a raise event with a deadline and add participants. Donors send sats to the event. When time runs out, the collected funds are distributed among all participants based on your configured split.

## Features

- Timed fundraising campaigns
- Multiple participant support
- Automatic fund distribution
- Public donation page

## Usage

1. Enable the extension in LNbits
2. Create a new raise event with a deadline
3. Add participants and configure the split
4. Share the event page
5. Funds distribute automatically when time expires
