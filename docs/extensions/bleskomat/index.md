## Overview

Bleskomat connects a [Bleskomat ATM](https://www.bleskomat.com/) (DIY or commercial) to your LNbits wallet. The ATM accepts fiat cash and displays an LNURL-withdraw QR code. Customers scan the QR with their Lightning wallet, and LNbits pays them from your wallet at the current exchange rate.

Since Bleskomat ATMs are designed to work offline, a cryptographic signing scheme verifies that the withdrawal URL was generated by an authorized device.

## Features

- **Offline ATM support** — ATM operates without an internet connection
- **Cryptographic verification** — signed LNURL-withdraw URLs prevent tampering
- **Fiat currency support** — configurable fiat currency matching your ATM
- **Exchange rate providers** — choose the API for fiat-to-sat conversion
- **Fee configuration** — set your ATM's fee percentage
- **Config export** — download a `bleskomat.conf` file for the ATM's SD card

## Setup

1. Enable the extension from the LNbits **Extensions** page
2. Click **Add Bleskomat**:
   - Choose the wallet to fund withdrawals
   - Select the fiat currency your ATM accepts
   - Pick an exchange rate provider
   - Set the fee percentage
3. Export the configuration file and copy it to your ATM's SD card
4. Restart the ATM — it loads the configuration automatically

## How It Works

1. Customer inserts fiat cash into the ATM
2. ATM generates a cryptographically signed LNURL-withdraw QR code
3. Customer scans with their Lightning wallet
4. Wallet contacts LNbits, which verifies the signature
5. LNbits converts the fiat amount to sats and pays the customer
