# LNbits SaaS

> Launch a fully managed LNbits instance in minutes — no server, no setup, no maintenance. Hosted by LNbits Inc.

[LNbits SaaS](https://saas.lnbits.com) is the official hosted solution for running LNbits without managing infrastructure. Instances deploy in under 3 minutes with full access to all features.

## Demo server

The demo server at [demo.lnbits.com](https://demo.lnbits.com) is for **testing only**.

::: danger
Do not store real funds on the demo server. Data is wiped frequently without notice.
:::

## LNbits SaaS

### How it works

1. Go to [saas.lnbits.com](https://saas.lnbits.com)
2. Choose a plan and pay (Lightning or card)
3. Your instance is live in under 3 minutes
4. Access your personal dashboard to manage everything

Each instance runs the same LNbits software as a self-hosted deployment — same extensions, same API, same features.

### Plans and pricing

Plans are available in flexible billing periods:

| Period | Best for |
| --- | --- |
| **Hourly** | Quick demos, workshops, testing |
| **Weekly** | Events, festivals, pop-up shops |
| **Monthly** | Ongoing projects, small businesses |
| **Yearly** | Production use, long-term deployments |

Pay with **Lightning** or **PayPal**. Visit [saas.lnbits.com](https://saas.lnbits.com) for current pricing.

### What you get

- **Full LNbits instance** — all extensions, all features, Admin UI access
- **HTTPS by default** — secure access out of the box
- **Management dashboard** — run multiple instances from one account for different use cases, clients, or projects
- **Data portability** — export your database and migrate to self-hosted anytime
- **Choose your funding source** — connect your own Lightning node, use an LSP, or run with FakeWallet for closed economies

### Use cases

- **Merchants** — accept Lightning payments with zero infrastructure
- **Events and festivals** — spin up a temporary instance with FakeWallet for internal economies
- **Schools and workshops** — teach Bitcoin and Lightning with real UIs and fake sats
- **Developers** — test extensions and integrations on a live instance
- **Multi-tenant setups** — manage separate instances for different clients from one dashboard

### Funding source

Each SaaS instance comes with a self-hosted **Spark L2** backend — a lightweight, self-custodial Lightning wallet with automatic channel management. No node setup required on your end.

If you need a different backend, you can switch to any supported funding source (LND, CLN, NWC, etc.) or use **FakeWallet** for closed economies like schools and festivals where no real money is needed.
